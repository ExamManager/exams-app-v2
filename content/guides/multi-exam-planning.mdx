## The problem

Exam weeks rarely run one paper at a time. Cohorts share halls, start times stagger, and staff need a single place to see what is live.

## What ExamManager planned for

- **Sessions** that hold several papers with their own start, end, and status.
- **Presets** for recurring subjects so you are not retyping durations under pressure.
- **Save-ahead windows** (by plan) so tomorrow’s papers can be prepared today.

## Suggested workflow

1. List every paper for the day with room and cohort.
2. Group overlapping papers into one session when the same invigilator covers them.
3. Confirm durations and any reading time before candidates enter.
4. Start each paper from the session list as the clock hits its start time.
