ExamManager V2 grouped timetable and seating helpers with the timer so staff could prepare the day before candidates arrive.

## Timetables

Build a readable list of papers with room, cohort, start time, and duration. Confirm overlaps early so invigilators know which sessions share a hall.

## Seating

Assign seats before the first paper. Candidates should know where to go without slowing the start of the timer.

## Suggested order of work

1. Lock the exam-week timetable.
2. Create seating for each room.
3. Create multi-exam sessions for overlapping papers.
4. Launch timers from the session list on the day.
