The classroom timer is the core of ExamManager V2: a large, readable countdown for invigilators, with optional student view.

<Callout type="warning">

This documentation is part of a **portfolio demo**. Auth and live sessions are disabled on this deployment.

</Callout>

## Controls

- **Start** — begin the countdown when candidates start writing.
- **Pause** — hold time when the room needs a brief interruption.
- **Extra time** — add minutes when rules allow it.
- **End** — mark the paper complete and clear the live state.

## Display tips

1. Keep the invigilator display uncluttered — status and remaining time only.
2. Use a projected student view when the hall needs a shared clock.
3. Prefer one primary timer per room; switch to multi-exam sessions when papers overlap.
