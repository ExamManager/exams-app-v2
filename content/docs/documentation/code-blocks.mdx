Pricing on ExamManager V2 matches the canonical ExamManager plans used across V1, landing, and V3. **Billing is disabled** — amounts are illustrative for the portfolio.

## Plans

| Plan | Price | Fit |
| --- | --- | --- |
| Standard | $0/mo | Personal use, up to 3 simultaneous exams, save 1 day ahead |
| Basic | $9.99/mo | Schools under 500 — planning, timetables, seating, presets (most popular) |
| Professional | $19.99/mo | Schools under 1000 — advanced planning and result analysis |
| Custom | from $39/mo | Larger capacity (contact via GitHub org — not offered here) |

Full feature lists live on the [Pricing](/pricing) page.

## Demo note

Signup, Stripe, and account dashboards are not available on this public deployment. Browse docs and guides, or visit [GitHub ExamManager](https://github.com/ExamManager) for source.
