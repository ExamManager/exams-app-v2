ExamManager V2 is an exam timing and classroom management app built for schools and invigilators. This documentation site is part of a **deprecated portfolio showcase** — accounts and billing are not active.

## What it covers

1. **Classroom timer** — large, low-distraction countdowns for exam halls.
2. **Multi-exam sessions** — several papers in parallel when rooms and cohorts overlap.
3. **Timetables** — readable schedules for exam weeks.
4. **Seating layouts** — plan where candidates sit before the timer starts.
5. **Plans** — illustrative Standard, Basic, and Professional tiers (see [Pricing](/pricing)).
